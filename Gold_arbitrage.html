Arbitrage between gold futures traded in Brazil and USA¶

Both Brazil's Bovespa Exchange and Chicago Commodities Exchange have future contracts for Gold. In Brazil, it is called OZ1D and in USA it is called GC. Since gold is a world commoditie, its price should be the same in all trade markets after FX effects. However, it is not always true. Although gold futures prices are correlated in both markets, on some days arbitrage opportunities occur because of difference in the markets, and it can be because of its size, on special player making a movement, and other technical aspects.

On this article, we will see how to identifiy these arbitrage opportunities using python. We start by importing all libraries we will need on our work

Now that all data are adjusted we can calculate a ratio between gold future contract in Brazil and in USA. The idea is that this ratio should keep around its mean, and every time it has a major deviation in opens up the opportunity for arbitrage, because it tends to return to its mean. So, we start by calculating the ration between Brazillian price and US price for each day of our data.

In [13]:
import statistics

#Calculating mean
mean_period = 72
div_mean = []

for i in range(0,len(GC_div_oz)):
    if i<mean_period:
        div_mean.append(0)
    else:
        div_mean.append(statistics.mean(GC_div_oz[i-21:i]))

#Calculating standard deviation
std = np.std(GC_div_oz)

mean_plus_dev = []
mean_minus_dev = []

for i in range(0,len(div_mean)):
    if div_mean[i]==0:
        mean_plus_dev.append(0)
        mean_minus_dev.append(0)
    else:
        mean_plus_dev.append(div_mean[i]+0.5*std)
        mean_minus_dev.append(div_mean[i]-0.5*std)

To make it more clear, we can create a data base with simulated operations with this contract pair. Every time the ration goes above one standard deviation from its mean, we would buy brazillian contract and sell american contract. The operation would be closed when the ration return to its mean.

    if Mean_data.iloc[i]['Mean']!=0:
        if start==0:
            if Mean_data.iloc[i]['GC / OZ']>Mean_data.iloc[i]['Mean + std dev']:
                dateStart.append(Mean_data.index[i])
                assetToBuy.append('OZ')
                priceAssetToBuy.append(Mean_data.iloc[i]['OZ price'])
                assetToSell.append('GC')

In this article we showed a simple analysis between gold future contracts traded in brazilian and US exchanges. There is further work to be done in order to make it a trading system, as calculating quantity ratios between the assets to build the operations. Also, there is some operational issues, as costs involved in operating a pair in exchanges from two different countries. Our objective here is to show that this arbitrage opportunity exists and how we can identify it using python dataframes and plotting tools.
